## 📌 프로젝트 소개

**AI 사주 천년지기**는 1000년의 지혜와 함께하는 명리학적 만세력과 AI 기술을 결합한 사주팔자 운세 서비스입니다.

### 핵심 가치
- 🎯 **정통 명리학**: 정확한 사주팔자 계산
- 🤖 **AI 상담**: 실시간 맞춤형 운세 상담
- 📱 **모바일 최적화**: 언제 어디서나 편리하게
- 🔒 **보안**: 안전한 개인정보 보호

## ✨ 주요 기능

### 🎴 사주팔자 서비스
- **무료 사주 보기**: 생년월일 기반 기본 사주팔자 분석
- **상세 사주 분석**: 일주 성격, 오행 분석, 2026년 운세
- **프리미엄 분석**: 대운 타임라인, 신살 분석, 용신 분석

### 💬 AI 상담
- **실시간 AI 채팅**: 무제한 질문 (회원가입 필요)
- **맞춤형 답변**: 개인 사주에 기반한 정확한 상담
- **추가 질문권**: ₩2,900 (10회)
- **프리미엄 구독**: ₩9,900/월 (무제한)

### 🔮 추가 서비스
- **궁합 분석**: 연애, 결혼, 사업 파트너 궁합 (₩9,900)
- **토정비결**: 2026년 12개월 월별 운세 (₩3,900)
- **택일**: 이사, 결혼 좋은 날 추천
- **꿈해몽 AI**: AI 기반 꿈 해석
- **작명/개명**: 성명학 기반 이름 추천
- **오늘의 운세**: 일일 운세 및 행운 정보
- **재물운 분석**: 생애 주기별 재물운 그래프
- **직업 적성**: 200+ 직업 DB 매칭
- **카카오 알람**: 매일 운세 푸시 알림

### 💰 추천인 시스템
- **추천인 보상**: 친구 유료 전환 시 ₩3,000 적립
- **피추천인 혜택**: 첫 구매 30% 할인
- **보너스**: 10명마다 ₩10,000 추가 지급
